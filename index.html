POWERED BY PURPOSE: ESHVA X MAHESH BABU FOUNDATION 3K RUNS

Together, we run to support children born with congenital heart ailments. This 3K Run, organized in collaboration with Eshva Celebrations and the Mahesh Babu Foundation, aims to raise awareness and strengthen access to life-saving heart treatments. Lace up, run with purpose, and help us create healthier futures for little hearts.
